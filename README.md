# Домашнее задание «Работа с PostgreSQL из Python»

## Задание

Необходимо создать программу для управления клиентами на Python.

Требуется хранить персональную информацию о клиентах:

- имя,
- фамилия,
- email,
- телефон.

Сложность в том, что телефон у клиента может быть не один, а два, три и даже больше. А может и вообще не быть телефона, например, он не захотел его оставлять.

Необходимо разработать структуру БД для хранения информации и несколько функций на Python для управления данными.

1. Функция, создающая структуру БД (таблицы).
2. Функция, позволяющая добавить нового клиента.
3. Функция, позволяющая добавить телефон для существующего клиента.
4. Функция, позволяющая изменить данные о клиенте.
5. Функция, позволяющая удалить телефон для существующего клиента.
6. Функция, позволяющая удалить существующего клиента.
7. Функция, позволяющая найти клиента по его данным: имени, фамилии, email или телефону.

Функции выше являются обязательными, но это не значит, что должны быть только они. При необходимости можно создавать дополнительные функции и классы.

Также необходимо предоставить код, демонстрирующий работу всех написанных функций.

## Результат

Результатом работы является `.py` файл.
